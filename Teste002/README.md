# 🚗 Dashboard de Vendas de Carros - Projeto Power BI - Teste 002

## 📌 Visão Geral

Este projeto tem como objetivo analisar dados de vendas de veículos utilizando Power BI, com foco no aprimoramento de métricas em DAX.

Diferente do projeto anterior, os dados foram extraídos de uma única base em Excel, permitindo maior foco na modelagem de medidas e cálculos analíticos.

---

## 🛠 Ferramentas Utilizadas

- Power BI
- Power Query
- DAX
- Excel

---

## 📊 Base de Dados

A base contém informações de vendas de veículos, incluindo:

- Modelo do carro
- Marca
- Data da venda
- Quantidade
- Preço
- Região / Localização

---

## 📈 Indicadores Criados (KPIs)

- Faturamento Total
- Quantidade de Veículos Vendidos
- Ticket Médio
- Vendas por Marca
- Vendas por Modelo
- Evolução das vendas ao longo do tempo

---

## 🧠 Foco do Projeto

Este projeto foi desenvolvido com ênfase em:

- Criação de medidas avançadas em DAX
- Uso de funções como:
  - CALCULATE
  - FILTER
  - DIVIDE
  - DATEADD
- Análise temporal
- Otimização de métricas

---

## 📊 Visualizações

O dashboard inclui:

- Cards com KPIs principais
- Gráfico de colunas (vendas por marca/modelo)
- Gráfico de linha (evolução temporal)
- Segmentações (filtros) por:
  - Marca
  - Período
  - Região

---

## 🎯 Objetivo do Projeto

- Evoluir habilidades em DAX
- Consolidar conhecimentos em Power BI
- Criar um dashboard analítico com base em dados reais
- Desenvolver portfólio na área de dados

---

## 🚀 Considerações

Este projeto representa uma evolução em relação ao anterior, com maior foco em lógica analítica e construção de indicadores.

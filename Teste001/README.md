# 📊 Dashboard de Vendas - Projeto Power BI - Teste 001

## 📌 Visão Geral

Este projeto tem como objetivo analisar dados de vendas provenientes de dois canais:
- Vendas Online
- Vendas Loja Física

Os dados foram tratados no Power Query, modelados em esquema estrela e visualizados no Power BI.

---

## 🛠 Ferramentas Utilizadas

- Power BI
- Power Query
- DAX
- Excel

---


---

## 📈 Indicadores Criados (KPIs)

- Faturamento Total
- Quantidade Vendida
- Ticket Médio
- Vendas por Produto
- Vendas por Canal
- Análise Temporal

---

## 🧠 Modelagem de Dados

Modelo dimensional (Esquema Estrela):

- Tabela Fato: `Total_Sales`
- Dimensão Produto: `Dim_Produto`
- (Possível Dimensão Calendário)

Relacionamento 1:* entre dimensão e fato.

---

## 📊 Visualização

O dashboard apresenta:

- Cards de KPI
- Gráfico de colunas por produto
- Gráfico de linhas temporal
- Segmentações por categoria e canal

---

## 🚀 Objetivo do Projeto

Projeto desenvolvido para fins de portfólio com foco em:

- Modelagem de dados
- Tratamento de dados
